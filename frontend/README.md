# SuccessFactors Configuration Bot - Frontend

## Overview
Modern React-based frontend for the SuccessFactors Configuration Bot.

## Features
- SuccessFactors authentication
- Workbook upload with drag & drop
- Version control visualization
- AI-powered analysis interface
- Configuration implementation dashboard

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at http://localhost:3000

## Build

```bash
npm run build
```

## Tech Stack
- React 18
- Vite
- React Router
- TanStack Query
- Zustand (State Management)
- Tailwind CSS
- Lucide Icons
